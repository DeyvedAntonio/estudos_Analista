# Semana 04 - Python básico
Dominar a sintaxe fundamental do Python necessária para desenvolvimento backend com Django, garantindo leitura, escrita e entendimento de código sem dependência de tutoriais.

## Tarefa
Criar scripts Python que resolvam problemas simples de decisão lógica e manipulação de dados básicos.

## Como executar
Considerando que já tem o Python instalado na sua máquina.

```
python script.py
```

## Quais conceitos de Python foram usados
Os principais conceito foram sobre manipulação de dados com uso de variáveis (que são espaços reservados na memória do computador), operadores relacionais e lógicos e funções built-in do Python como int(), input() e print()
