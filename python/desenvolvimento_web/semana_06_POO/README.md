# Contexto do Caso

Instituições financeiras precisam avaliar o risco de conceder crédito a um cliente antes de aprovar empréstimos, financiamentos ou cartões.

Essa decisão se baseia em dados cadastrais, financeiros e comportamentais.

## Entidade Central do Sistema

Entidade: **AnaliseCredito**

Essa entidade representa uma avaliação de risco realizada para um cliente em um determinado momento.
